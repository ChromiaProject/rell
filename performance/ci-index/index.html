Rell Quality Reports

CI jobs may publish reports for three suites: Benchmarks (JMH microbenchmarks), Profile (end-to-end async-profiler run against a Chromia node + workload), and Regression (clone a curated list of external Rell projects and verify whether the snapshot Rell is still valid in them). Benchmark and profile deployments also expose JSON at <deployment>/data/main.json — the compare tool below fetches two profile runs and diffs them client-side.
