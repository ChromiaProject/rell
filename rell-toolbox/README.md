# Rell Toolbox

Tools for the Rell programming language.

* [Rell Tools Core](docs/Core.md) - Core library for Rell tools (e.g. Parser, AST Converter, Indexer etc.)
* [Rell Tools Formatter](docs/Formatter.md) - Library for formatting of Rell source code
* [Rell Language Server](docs/LanguageServer.md) - Library for Language Server used for IDE support

See [DEVELOPMENT.md](../DEVELOPMENT.md) for building, testing, and release instructions.
